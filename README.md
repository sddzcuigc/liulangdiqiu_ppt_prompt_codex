# 《流浪地球》网页 PPT 项目提示词总结

## 一次成型的完整提示词

你要帮我制作一个《流浪地球》主题的网页版 PPT，适合小学三年级学生做读书分享。请直接完成，不要只给方案。

要求如下：

1. 形式与输出

- 做成一个本地可直接打开的网页版 PPT。
- 页面要适合全屏展示，支持左右切页、自动播放、自动翻页、鼠标点击翻页、滚轮翻页。
- 要有一个本地启动脚本，方便直接运行。
- 不要只做静态页面，要做成真正可演示的作品。

2. 内容来源与文案要求

- 以我提供的《流浪地球》读书分享原文为基础。
- 保留原文核心意思，但可以根据演讲需要做适度润色。
- 如果我要求“全文保留”或“不要删减”，就必须只分页、不改写。
- 如果我要求做三年级版本，就要把语言顺得更口语、更自然，更适合小学生上台分享。
- 文案要体现：
  - 对《流浪地球》故事的理解
  - 对“希望、理性、信仰、坚守”的主题表达
  - 一定程度的科学素养，比如近日点、太阳引力弹弓、重聚变发动机、五个阶段等设定

3. 页面结构

- 首页做正式封面页。

- 封面页内容包含：

  - 《流浪地球》读书分享
  - 宁津县第二实验小学 3年级6班
  - 崔护
  - 2026年4月

- 首页背景优先使用《流浪地球》海报。

- 首页显示内容和朗读内容可以分开：

  - 页面显示为正式封面
  - 自动朗读内容为自我介绍

- 自我介绍朗读词为：

  我是宁津县第二实验小学 3年级6班 崔护，今天很高兴在这里分享我喜爱的一部科幻作品，刘慈欣的《流浪地球》。

4. 背景素材要求

- 不必所有页面都用视频。
- 有合适视频的页面用视频。
- 没有合适视频的页面用图片。
- 如果视频播放失败，要自动回退到图片，不要黑屏。
- 图片页要有轻微动态效果，像电影海报在缓慢运动，不要完全静止。
- 蒙版不要太黑，要能调亮度。

5. 视频素材处理要求

- 你可以帮我搜索、筛选、下载、裁剪视频。
- 需要根据我给的 B 站链接，截取指定时间段，保存到本地，再替换到指定页。
- 例如：
  - 某条视频截取 6:12 到 7:17 放到第 2 页
  - 某条视频截取前 5 秒放到某页
- 视频要尽量和页面内容匹配，比如：
  - 开场适合暗红天空、灼热太阳、近日点场景
  - 设定页适合行星发动机
  - 冲突页适合压迫感强的镜头
  - 高潮页适合爆发力强的镜头
- 如果找不到足够多的视频，可以混合使用图片。

6. 图片素材要求

- 你可以帮我搜索、下载、整理图片。
- 图片要按页对应叙事内容。
- 最后的致谢页可以根据要求切换素材，比如：
  - 刘慈欣照片
  - 刘慈欣签售照
  - 《流浪地球》实体书封面
  - 书店里摆放的《流浪地球》实体书实拍图

7. 音频与朗读

- 页面要支持背景音乐播放。
- 要支持音量调节。
- 要支持自动朗读。
- 优先调用 Windows 中文语音“晓晓 / Xiaoxiao”。
- 自动朗读要读我指定的稿子，而不是只读标题。
- 自动翻页应尽量适配文字长度和朗读时长，不要每页固定死时间。

8. 交互要求

- 支持自动播放朗读。
- 支持播放/暂停视频。
- 支持播放/暂停音乐。
- 支持全屏。
- 支持鼠标点击左右区域翻页。
- 支持滚轮翻页。
- 支持显示计时器。
- 计时器至少包括：
  - 总时长
  - 当前页时长

9. 页面显示要求

- 页面上不要出现给我看的制作注释、素材说明、调试信息。
- 界面尽量干净，适合正式展示。
- 页面顶部标签可以改成“崔护制作”。

10. 文案风格要求

- 整体语气要统一。
- 要像一篇完整的演讲稿，而不是几段拼起来的话。
- 能体现小学生的理解，但不要太幼稚。
- 有科学感，但不要太学术。
- 有感情，但不要太空泛。
- 要多用自然递进，而不是突然拔高。

11. 文件管理要求

- 如果我要求“不要改原作品”，你必须保留原文件。
- 新的润色版、低龄版、三年级版要另存副本。
- 文件命名清楚，方便我直接打开使用。

12. 最终交付要求

- 直接把网页、素材、启动脚本都做好。
- 告诉我入口文件在哪里。
- 如果有多个版本，说明每个版本用途。

## 适合直接复用的短版提示词

请帮我制作一个《流浪地球》主题的网页版读书分享 PPT，适合小学三年级学生演讲。要求直接完成网页、素材整理、视频裁剪、自动朗读、自动翻页、背景音乐、鼠标翻页和计时器，不要只给方案。页面支持视频与图片混合背景，有视频用视频，没有视频用图片，图片要有轻微动态效果。文案基于我提供的原文进行润色，但保留核心意思；如果我要求保留原文，就只分页不改写。首页做正式封面页，朗读词可与显示内容分开。支持调用中文语音“晓晓”朗读。最后输出本地可直接运行的 html 和启动脚本，并在不覆盖原文件的前提下另存不同版本副本。

## Codex 在这个项目里可以自己做的事情

### 1. 文案整理

- 根据原文拆分页面结构
- 对文案做润色、顺句、降龄化表达
- 做“正式版 / 三年级版 / 原文保留版”等多个副本
- 调整标题、页间衔接、结尾收束语言

### 2. 页面开发

- 编写 HTML / CSS / JavaScript
- 制作全屏网页版 PPT
- 实现自动翻页、自动播放、鼠标翻页、滚轮翻页、全屏、计时器等功能
- 实现视频页和图片页混合逻辑
- 实现图片动态效果和蒙版亮度调节

### 3. 语音与音频

- 接入浏览器 `SpeechSynthesis`
- 优先选择系统里的“晓晓 / Xiaoxiao”语音
- 让朗读按页面内容或单独 `speechText` 播放
- 接入背景音乐、本地音频文件、音量控制

### 4. 素材搜集

- 上网搜索图片和视频素材
- 选择更适合当前页面情绪的素材
- 下载到本地并整理命名
- 将素材自动接入页面

### 5. 视频处理

- 根据 B 站链接提取可下载媒体流
- 下载视频与音频流
- 合并视频音频
- 裁剪指定时间段
- 重新编码为网页兼容的 MP4
- 将裁剪后的视频替换到指定页面

### 6. 文件组织

- 创建新副本，不覆盖原作品
- 生成启动脚本
- 把所有素材统一整理到 `assets/images`、`assets/video`、`assets/audio`
- 产出可直接交付使用的目录结构

### 7. 可继续扩展的事情

- 再生成一个“比赛汇报版”
- 再生成一个“更电影感”的版本
- 自动隐藏控制栏，做纯演示模式
- 按实际朗读时长进一步微调翻页节奏
- 替换不同风格的海报、书封、签售图、作者照

## 使用建议

- 如果以后还要继续做别的读书分享，只要把“主题、原文、学校班级、作者、音乐、指定视频链接、页面风格要求”替换掉，这套提示词基本可以继续复用。
- 如果你已经有完整文稿，最好在提示词里明确：
  - 哪些句子必须保留
  - 哪些句子允许润色
  - 是否允许合并页面
  - 是否允许副本另存
